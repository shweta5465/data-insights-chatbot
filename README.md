# 📊 Data Insights Chatbot

An AI powered chatbot that answers questions about 
sales data and automatically generates charts!

## 🛠️ Built With
- Python
- Streamlit
- Groq AI API
- Pandas
- Plotly

## 🚀 Features
- Natural language data querying
- Auto chart generation for every question
- Real Superstore dataset with 9994 rows
- Beautiful chat interface

## ▶️ How to Run
1. Clone this repo
2. Install requirements:
pip install -r requirements.txt
3. Create .env file and add your Groq API key:
GROQ_API_KEY=your_key_here
4. Run:
streamlit run app.py
